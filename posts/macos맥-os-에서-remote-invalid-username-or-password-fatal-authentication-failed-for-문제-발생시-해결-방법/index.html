macOS(맥 OS) 에서 remote: Invalid username or password. fatal: Authentication failed for 문제 발생시 해결 방법

Posted Jan 12, 2021
By ayaysir
1 min read

깃허브에 문제가 생겨서 인증 에러(Authentication failed)가 발생한 경우 아래와 같은 방법으로 해결할 수 있습니다. (macOS 기준, 윈도우는 링크 참조)

 

1. Spotlight 검색창 또는 Launchpad > 기타 폴더에서 ‘키체인 접근’이라는 응용프로그램을 실행합니다.

 

2. 검색창에 git을 검색한 뒤 ‘인터넷 암호’ 종류의 github.com 항목을 삭제합니다.

 

3. 다시 push를 시도하면 아이디, 비밀번호 입력창이 나오는데 정확하게 입력합니다.

 

이렇게 하면 github 계정 정보가 갱신되면서 인증 오류를 해결할 수 있습니다.
